📋 ProFlow
All projects, one flow.
A simple project and task management system designed for developers and small teams.

🚀 About The Project
ProFlow is a web application for managing projects and tasks, built to organize daily work and track progress without complexity.
The project is built with Laravel on the backend and React + TypeScript on the frontend, with a modern design using Material UI.

✨ Features
✅ Manage multiple projects

✅ Add, edit, and delete tasks

✅ Track progress and productivity

✅ Interactive statistics and charts

✅ Clean and easy-to-use interface

✅ Fast and modern UI
